# Smart Delivery Routing Platform

## Project Intro
Smart Delivery Routing Platform is a city-scale delivery coordination system for managing orders, assigning dispatches, and estimating route and ETA between hubs.

## Run Frontend
```bash
cd frontend
npm install
npm run dev
```

## Run Backend
```bash
cd backend
mvn spring-boot:run
```
